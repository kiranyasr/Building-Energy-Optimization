User Guide

How to use the Building Energy Optimization application.

Welcome!

This application is a full-stack data science project demonstrating an end-to-end machine learning pipeline for building energy analysis.

The Analysis Pipeline

Follow the steps in the sidebar navigation to proceed through the pipeline:

  1. Upload Data: Begin by uploading your energy consumption dataset in CSV format.
  2. Clean & Preprocess: Automatically handle missing values, duplicates, and outliers.
  3. Fluctuation Analysis: Visualize trends and volatility in the time-series data.
  4. Feature Engineering: Create new, informative features like time lags and moving averages.
  5. Feature Selection: Use machine learning to identify the most important predictive features.
  6. Baseline Models: Train several standard machine learning models as a performance baseline.
  7. Deep Learning: Train advanced sequence models (LSTMs) for potentially higher accuracy.
  8. Optimization: Fine-tune the best model's hyperparameters to maximize its performance.
  9. Anomaly Detection: Use the final model to find unusual spikes or dips in consumption.
  10. Wastage Analysis: Identify periods of energy wastage and get actionable insights.
  11. Final Report: View a summary of all models and download project artifacts.

How to Use the Deployment Page

Navigate to the 'Deployment' tab. This page allows you to use the final, optimized model to make live predictions. Fill in the values for each required feature in the form and click "Predict Consumption" to get a real-time prediction for the energy consumption.
